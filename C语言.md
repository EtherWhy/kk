# C & C++

- C语言1972年诞生
         - 贴近硬件，运行极快，效率极高。
         - 操作系统，编译器，数据库，网络系统等。
         - 语法、变量限制不严格，数据安全
         - 指针和内存管理。(你不对指针赋初值，指针可能会指向一个不确定的地方，有可能是关键进程或者是系统地址，这就导致了程序或者系统崩溃。)
         - 同时期的**Unix**系统配合C语写了很多操作系统

- C++1982年诞生
  - 面向对象
  - 兼容C
  - 图形领域、游戏等。
  - 计算机底层支持。

- Java
  - 语言有点像C
  - 没有指针
  - 没有内存管理
  - 真正的可移植性，编写一次，到处运行。
  - 面向对象
  - 类型安全
  - 高质量的类库
