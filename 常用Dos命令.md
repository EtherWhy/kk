# 常用Dos命令

```base
#显示当前目录下的所有文件：dir
#切换目录：cd change directory
#返回上一级：cd..
#清理屏幕:cls clean screen
#退出终端：exit
#查看电脑IP:ipconfig
#查看网络质量:ping www.baidu.com
			ping
            223.5.5.5\223.6.6.6(阿里)
			119.29.29.29\182.254.116.116(腾讯)
			180.76.76.76(百度)
#打开电脑应用：
	calc(计算器)
	mspaint(画图软件)
	notepad(记事本)
#文件操作：
	md 目录名(创建目录)
	rd 目录名(删除目录)
	cd> 文件名(创建文件)
	del 文件名(删除文件)
			
```
